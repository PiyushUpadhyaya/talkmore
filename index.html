TalkMore looks at your call, message and mobile data usage to recommend the right set of mobile plans. You can find the best plans by estimating future usage. This application is for mobile phone users in India.


To start using, install the app and setup your mobile operator and circle. You can immediately start looking at plan recommendations based on your usage. If you expect future usage to be different, make changes in the usage sliders and get new recommendations. You can click on a recommendation to get more details.

This is an early release and any feedback will drive future product releases.

We currently support most leading mobile operators in India (Airtel, Idea, Reliance, Vodafone, BSNL).

The application runs completely locally and no private information is recorded.

Recharging from the app is coming soon. Until then, recharge from a store or online using the cost given in the header in each voucher.
